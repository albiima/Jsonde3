Overview
==========

jSonde is a brand new tool for profiling, analyzing and reverse engineering Java applications.
It allows you to automatically generate UML Sequence Diagrams of your applications,
profile processor and memory usage, build various reports and many other things.

Features
==========

 * Attach to running Java processes without restarting them.
 * Generate sequence diagrams.
 * Profile method calls.
 * Profile memory usage.
 * Show .jar files dependencies.
